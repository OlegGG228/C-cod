# a) Для печати всех неотрицательных элементов массива, можно использовать следующий код:

```python
arr = [1, -2, 3, -4, 5, 6]
for x in arr:
    if x >= 0:
        print(x)
```
# б) Для печати всех элементов, не превышающих число 100, можно использовать следующий код:

```python
arr = [20, 50, 120, 80, 90, 100]
for x in arr:
    if x <= 100:
        print(x)
