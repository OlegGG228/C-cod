# a) Для ладьи условием является то, что она стоит на одной вертикали или горизонтали с полем (с, d).
```python
if a == c or b == d:
    print("Ладья угрожает полю ({}, {})".format(c, d))
```

# б) Для слона условием является то, что он стоит на одной диагонали с полем (с, d).
```python
if abs(a - c) == abs(b - d):
    print("Слон угрожает полю ({}, {})".format(c, d))
```

# в) Для короля условием является то, что расстояние между его полем и полем (с, d) не превышает 1 по вертикали, горизонтали или диагонали.
```python
if abs(a - c) <= 1 and abs(b - d) <= 1:
    print("Король может попасть на поле ({}, {}) одним ходом".format(c, d))
```

# г) Для ферзя условием является выполнение условий для ладьи или слона.
```python
if a == c or b == d or abs(a - c) == abs(b - d):
    print("Ферзь угрожает полю ({}, {})".format(c, d))
```

# ж) Для коня условием является то, что он стоит на одной из 8 возможных клеток, смещенных от поля (c, d) на определенное расстояние по вертикали и горизонтали.
```python
if (abs(a - c) == 2 and abs(b - d) == 1) or (abs(a - c) == 1 and abs(b - d) == 2):
    print("Конь угрожает полю ({}, {})".format(c, d))
