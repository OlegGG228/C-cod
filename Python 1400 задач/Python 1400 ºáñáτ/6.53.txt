а) Для определения наличия цифры а в числе, можно преобразовать число в строку и проверить наличие символа а в строке:

num = 77
a = '7'
if a in str(num):
    print('Цифра а есть в числе')
else:
    print('Цифры а нет в числе')

б) Для проверки отсутствия цифры Ь в числе, можно использовать тот же подход:

num = 77
b = '8'
if b not in str(num):
    print('Цифры Ь нет в числе')
else:
    print('Цифра Ь есть в числе')

в) Для определения количества вхождений цифры а в числе, можно также преобразовать число в строку и использовать метод count:

num = 77
a = '7'
k = 2
if str(num).count(a) > k:
    print(f'Цифра {a} встречается в числе более {k} раз')
else:
    print(f'Цифра {a} встречается в числе не более {k} раз')

г) Для проверки наличия цифр а и Ь в числе, можно использовать операторы in и not in:

num = 77
a = '7'
b = '8'
if a in str(num) and b in str(num):
    print('Цифры а и b есть в числе')
else:
    print('Цифры а и/или Ь нет в числе')
