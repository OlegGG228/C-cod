a) Урожайность за второй, третий, ..., восьмой год:
```
урож1 = 20 # урожай в первый год
for i in range(2,9):
    урож2 = урож1 * 1.02 # урожай в следующий год
    print("Урожайность за", i, "год:", round(урож2, 2))
    урож1 = урож2 # запоминаем урожай для следующего года
```
б) Площадь участка в четвертый, пятый, ..., седьмой год:
```
площадь1 = 100 # площадь в первый год
for i in range(2,8):
    площадь2 = площадь1 * 1.05 # площадь в следующий год
    if i in [4, 5, 6, 7]: # выводим только для нужных лет
        print("Площадь участка в", i, "год:", round(площадь2, 2))
    площадь1 = площадь2 # запоминаем площадь для следующего года
```
в) Урожай за первые шесть лет:
```
урож1 = 20 # урожай в первый год
площадь = 100 # площадь в первый год
урожай_итого = урож1 * площадь # урожай за первый год
for i in range(2,7): # только первые 6 лет
    урожай = урож1 * площадь * 1.02**i # урожай в текущем году
    урожай_итого += урожай # добавляем текущий урожай к общему
print("Урожай за первые шесть лет:", round(урожай_итого, 2))
