Решение 1 (с использованием оператора цикла с условием):

```python
p = int(input("Введите число p: "))
n = 1
while n * n <= p:
    n += 1
print("Первое число, большее", p, "равно", n*n)
```

Решение 2 (без использования оператора цикла с условием):

```python
import math
p = int(input("Введите число p: "))
n = int(math.sqrt(p)) + 1
print("Первое число, большее", p, "равно", n*n)
```

Оба решения используют функцию math.sqrt() для вычисления квадратного корня числа p. Второе решение также использует функцию int() для округления результата к целому числу.
